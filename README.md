##### 使用axios+vite+Es6构建油价搜索查询功能

###### 使用方法：

> 克隆项目到本地
>
> ```
> git clone 仓库地址
> ```
>
> 进入项目根目录，安装依赖
>
> ```
> npm install
> ```
>
> 启动项目
>
> ```
> npm run dev
> ```
>
> 在浏览器打开
>
> ```
> localhost:7000
> ```

###### 此项目为练习项目，实际意义不大，主要是为了练习使用如下技术：

>axios以及axios拦截器
>
>使用vite快速构建和打包项目
>
>ES6模块化编程以及部分ES6新特性

###### 后台接口由[优客API](https://api.iyk0.com)提供。
